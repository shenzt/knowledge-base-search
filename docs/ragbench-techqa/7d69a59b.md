Oracle; 12c TECHNOTE (FAQ)

QUESTION
 What versions of Tivoli Workload Scheduler (TWS) are supported with Oracle 12c?

ANSWER
Oracle 12c is supported with IBM Workload Scheduler (IWS) v9.1, v9.2, and v9.3.

NOTE: You can check the compatibility of TWS with various components by using the Software Product Compatibility Reports (SPCR) website:
http://www.ibm.com/software/reports/compatibility/clarity/index.html [http://www.ibm.com/software/reports/compatibility/clarity/index.html]
